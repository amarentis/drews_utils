# ansible-tools.sh
Creates new ansible scaffolding 
* create a directory
* cd into directory & git init
* ansible-tools.sh create mynewplaybook
* vagrant up
* start adding more tasks in roles/mynewplaybook/tasks/main.yml
* vagrant provision to run those changes
